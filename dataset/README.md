# Dataset Description

This dataset contains mobile sales data used to build the Power BI dashboard.

## File Name
- Mobile Sales Data.xlsx

## Data Description
The dataset includes the following fields:
- Transaction ID
- Day
- Month
- Year
- Day Name
- Brand
- Units Sold
- Price Per Unit
- Customer Name
- Customer Age
- City
- Payment Method
- Customer Ratings
- Mobile Model


## Purpose
This dataset is used to analyze:
- Sales performance over time
- Quarterly trends
- Regional sales distribution
- Product-level insights
